# Introduction 
> I made this config for using on my Android device(Cause I am broke).
> \
> So I made my first config of Neovim from a post in dev.com and used it for a while. Unfortunately, My device had an issue which caused me to lose all my Configs.
> \
> And after losing my configs twice I wanted to upload it to *GitHub* so that I don't lose it again.

Thus I ended up making this config. i have read a lot of guides on how to make a config properly but most of them required some type of knowledge with these things which I *didn't* have. 

So I have tried to make it as easy to understand as possible so others who want to try it can use these files and make their own configs easily.

I hope others who just started their *Neovim* journey would find this useful.

# Features
Key Features
* Readable Plugins file(**plugins.lua**).
* Useful *comments* to help other understand what each options do.
* Everything required for Web Development *out-of-the-box*.
* A ton of *Themes* and a *Theme Switcher* to use them easily.

\
Additional Features
* **Animated Statusline** with custom animation file that shows how to make your own animations(**work in progress**).
* **Notifications**.
* Useful **keymaps** and auto **key suggestions** to work with.
* Helper function to easily make **custom** keymaps.
* Touch + keyboard controlled **tab switcher**.
* Auto line **centering**.
* **File viewer** that can do Creating, Renaming, Previewing, copying etc.
* Easier Command line suggestions(No need to use **Tab** anymore)
* Option to **dim** parts of Code that you aren't editing.


And a lot more!
